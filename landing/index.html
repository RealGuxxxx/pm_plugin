快速配置

5分钟完成部署，开启智能交易

Polymarket AI Agent 是一款开发中的解压插件，您只需将其克隆至本地并在 Chrome 中加载，运行一个轻量级的本地桥接服务，即可解锁完全体交易助手。

1

克隆或获取代码仓

将当前插件的代码目录保存在本地计算机上，准备加载入浏览器。

2

以“开发者模式”载入 Chrome

打开 Chrome，访问 chrome://extensions/，勾选右上角的“开发者模式”，点击“加载已解压的扩展程序”，选择插件文件夹即可。

3

运行 xapi 本地桥接服务

在插件文件夹目录下打开终端，安装并运行 Node 桥接守护进程。该程序将开启 3000 端口，用于安全调用 xapi 舆情搜索与 CLI 服务。

node xapi_daemon.js
4

激活 Gemini API 密钥

点击浏览器右上角插件图标，进入控制台 (Dashboard)，输入您的 Gemini API 密钥（用于社交舆论的情感极性提炼）即可完全激活。

bash - xapi_daemon
$node xapi_daemon.js
